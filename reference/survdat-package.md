# survdat: Pull and process Survey data

Tools to aid in extraction of survey data from the svdbs database

## Details

- Pulls bottom trawl survey data from svdbs

- Applies corrections to the data based on changes in gear and ships

- Partitions the data based on user supplied geospatial data

- Estimates biomass of species

- Helper functions to explain species, maturity, strata,vessel, sex
  codes

- Pulls length-weight data and/or coefficients
